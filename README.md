# review-rot

Currently the tool includes listing pull reuqests for github, gitlab and pagure.

### Sample I/P:
create '~/.config.yaml' with the content from sampleinput.yaml

### Dependencies:

Sudo pip install PyGithub  
pip install --upgrade python-gitlab

### Run:
python reviewrot.py
